BrowserRamCache v0.1.3
By Sriranga Veeraraghavan <ranga@calalum.org>

BrowserRamCache is a perl script that moves Safari, Chrome, and Opera's
disk cache to a ram disk.  The cache is not synced to disk or preserved
on reboot.

To install:

1. make install
2. reboot

History:

v. 0.1.3 (06/28/2017) - Add support for Opera
v. 0.1.2 (04/24/2013) - Reduce default cache size to 16MB
v. 0.1.1 (03/27/2011) - Fix unmount bug, secure permissions for cache
                        directories
v. 0.1.0 (03/26/2011) - Initial Release
